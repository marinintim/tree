tree
====

Simple tree utility.

```sh
$ tree test
┬
└┬/Users/mt/Workspace/src/github.com/marinintim/tree/test
 ├┬dir1
 │├─sub1
 │├┬sub2
 ││├─sub1
 ││├─sub2
 ││└─sub3
 │└─sub3
 └─dir2
```

See [DESIGN.md](https://github.com/marinintim/tree/blob/master/DESIGN.md) for more details.
